# Snap Stock

## Backend

- Python 3.10.10
- Flask 3.0.0

## Frontend

- HTML5
- CSS3
- JavaScript

## Instructions

- Clone repository

```bash
git clone https://github.com/teanirudh/snap-stock.git
```

- Change directory

```bash
cd snap-stock
```

- Install Flask

```bash
pip install Flask
```

- Run the server

```bash
python app.py
```
